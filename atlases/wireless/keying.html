bits → symbols → { amplitude | frequency | phase }

Amplitude & frequency keying

01 / OOK

I envelope OOK — On-Off Keying

s(t) = b(t) · cos(2π f_c t), b ∈ {0, 1}

AM QAM RFID / IoT
02 / ASK

I envelope ASK — Amplitude Shift Keying

s(t) = A_k · cos(2π f_c t), A_k ∈ {A₀, A₁, …, A_{M−1}}

OOK QAM
03 / FSK

I discrim. FSK — Frequency Shift Keying

s(t) = cos( 2π f_c t + 2π h ∫ a(τ) dτ ), h = modulation index

FM GFSK GMSK Bluetooth LE
04 / GFSK

II coherent GFSK — Gaussian FSK

g(t) = Gaussian( BT ) ∗ rect, smaller BT ⇒ narrower spectrum, more ISI

FSK GMSK Bluetooth DECT
05 / M-ary FSK

II discrim. 2-FSK / 4-FSK

4-FSK: dibit → one of 4 tones {−3Δ, −Δ, +Δ, +3Δ}

FSK DECT Paging
06 / MSK

II coherent MSK — Minimum Shift Keying

h = 0.5 (the smallest index giving orthogonal, continuous-phase tones)

FSK GMSK O-QPSK
07 / GMSK

II coherent GMSK — Gaussian MSK

GMSK = MSK + Gaussian pre-filter (GSM uses BT = 0.3)

Where it is used
MSK GFSK GSM 8-PSK / EDGE

Phase-shift keying (PSK)

08 / BPSK

I coherent BPSK — Binary PSK

s(t) = cos(2π f_c t + π·b), b ∈ {0,1} → two antipodal points (0°, 180°)

PM GPS QAM
09 / QPSK

II coherent QPSK — Quadrature PSK

4 phases (±45°, ±135°) · 2 bits/symbol · same BER as BPSK for same Eb/N0

BPSK O-QPSK UMTS/3G DVB-S
10 / O-QPSK

II coherent O-QPSK — Offset QPSK

delay Q by half a symbol ⇒ no 180° transitions ⇒ smaller envelope dips

QPSK MSK ZigBee / Thread
11 / π/4-DQPSK

II differ. π/4-DQPSK

every symbol rotates the constellation by ±π/4 ⇒ 8 phase states, never through the origin

QPSK DPSK TETRA
12 / 8-PSK

II coherent 8-PSK

8 phases · 3 bits/symbol · ~3.6 dB less noise margin than QPSK

QPSK 16-QAM EDGE
13 / DPSK

I non-coh. DPSK — Differential PSK

encode: dₖ = dₖ₋₁ ⊕ bₖ (as phase); decode: b̂ₖ = phase(dₖ) ⊕ phase(dₖ₋₁)
